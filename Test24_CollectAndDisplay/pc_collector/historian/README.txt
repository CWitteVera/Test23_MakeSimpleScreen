This directory stores historian CSV files generated by collector.py.

One file is created per calendar day (UTC):
    conveyor_history_YYYY-MM-DD.csv

Each row contains:
    timestamp_utc, L1Z1, L1Z2, L1Z3, L2Z1, L2Z2, L2Z3, L3Z1, L3Z2, L3Z3, total_in_system

These files are intentionally excluded from version control (.gitignore).
Add them to your archive / backup solution as appropriate.
